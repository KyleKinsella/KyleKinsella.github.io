Algorithms Online & Sources

During the development of my support vector machine project I found an online source that was some help to me.

ChatGpt: ChatGPT

The above website was a little helpful for this project. I used chatgpt, to try and find out why was my decision boundary line not discriminating between the two classes. I asked him something like: how do I draw a line that separates my two classes.


The reason why my support vector machine model was not discriminating between my two classes was because I was not using a Logistic Regression model. So I implemented a Logistic Regression model to my code and eventually I got it to work.

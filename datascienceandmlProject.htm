Naïve Bayes

During this project, I explored Naïve Bayes. Naïve Bayes is a supervised learning machine learning model. Naïve Bayes is a generative learning algorithm, this means the model looks for the inputs of a given class or category (Naïve Bayes). There are three types of Naïve Bayes (See below). Types of Naïve Bayes.

  • Gaussian - works on continuous data
  • Multinomial - works on discrete data
  • Bernoulli - works on binary/boolean data

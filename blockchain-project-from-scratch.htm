Why am I doing this?

The goal of this project is to gain a deeper understanding on how a blockchain works under the hood.

Technologies

I have chosen to build this project with the following tech stack:

Frontend

  • React

I chose React because it's a widely used framework in modern software development and I want to strengthen my skills with React by applying it to this project.

Backend

  • Go (Golang)
  • Might use Python later in the project

I have chose to develop my backend in Go because Go contains concurrent features, such as: Go-Routines, Wait-Groups, Mutex Locks and Channels. Another reason to why I chose to build my backend in Go is because Go is a fun programming language to write software in.

What have I made so far?

So far, I have developed most of the core features that a blockchain has, such as: Blocks, Proof of Work (POW) and Proof of Stake algorithms, Transactions, Public & Private Key-Pairs and Addresses.

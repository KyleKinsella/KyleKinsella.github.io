Data Sources

Before I started to work on my recurrent neural network project, I found a dataset that I wanted to use for my recurrent neural network project. The dataset that I used for this project is not from kaggle, it is a dataset that is in the tensorflow library. The dataset is called: tf.keras.datasets.mnist.load_data, the dataset has images of numbers in training and test sets. For my recurrent neural network project I wanted to create a machine learning model to be able to learn different types of number images. And to try and train a recurrent neural network to be able to identify the different types of numbers. This is the dataset that I used for this project: Tensorflow Dataset.

Pre Processing

During my recurrent neural network project, I did pre-process my dataset. After I imported the mnist dataset, I split the dataset into training and test sets. I also then normalized X_train and X_test.

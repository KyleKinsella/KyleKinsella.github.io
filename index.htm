Random Test Data Generation (2023)

Java • Unit Testing • Software Engineering • Documentation

What is Random Test Data Generation?
Random Test Data Generation is a Java application designed to automatically generate unit tests by producing randomized input data. This helps ensure robust testing coverage and reduces the manual effort required when writing test cases. Below are some of the key aspects of the Random Test Data Generation project:

  • Wrote formal documents including: Vision Document, Specification Document, Research & Design Document, Project Report and Iteration Reports.

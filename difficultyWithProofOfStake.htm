Proof of Stake Algorithm

In this document, I will be outlining the issues that I had with trying to implement my very own Proof of Stake (POS) algorithm.

Note: all of this code is wrote in Go.

Problem One

The first problem that I encountered was the following: I was trying to use a block struct that I have created for my Proof of Work (POW) Algorithm. Below is how my Block looks like: 


        type Block struct {
            Index int
            Timestamp string 
            PrevHash interface{}
            Transactions Transaction
            ProofOfWork bool 
            BlockHash string 
        }

        type Transaction struct {
            Sender string
            Receiver string
            Amount float32
        }

The above code is the Syntax for making a block for a POW Blockchain. The problem then arrised when I was coming back to my POS algorithm, because I was trying to have the above syntax for a POS block - this is entirely wrong!

Below is the code that I previously wrote for the start of my POS algorithm / code:


    type Block struct {
        Index int
        Timestamp string 
        PrevHash interface{}
        Transactions Transaction
        ProofOfWork bool 
        BlockHash string 
    }

    type Validator struct {
        block Block
    }

    func (v *Validator) ValidateBlock(block *Block) bool {
    }

After doing some analysis on my code, I realized one crucial problem. A validator has a Block type as a field. But: a Block has a "ProofOfWork" field, I am implementing a Proof of Stake algorithm. So I had to abandon this code - simply because it is completly useless!

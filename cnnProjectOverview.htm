Before I started to work on my convolutional neural network project, I found a dataset that I wanted to use for my convolutional neural network project. For my convolutional neural network project I wanted to create a machine learning model to be able to learn different types of fruit images. And to try and train a convolutional neural network to be able to identify the different types of fruits. This is the dataset that I used for this project Fruit Classification Dataset.

Pre Processing

During my convolutional neural network project, I did pre-process my dataset. I changed the size of each image from 100 x 100 to be 25 x 25. I also used batch_size=32, this means that the images are in batches(groups), this makes it easier to process while in the training of the model. I also split my data into training and validation sets. I also am using seed=123, this means that each time the code runs the split is different each time.

Pre Processing

During this project on k means clustering I did some pre-processing. During this phase of my k means clustering project, I was plotting my data to a graph for visualization, but when I was doing this I did realize that my data was very spread out. So I used something called a MinMaxScaler. When I used the MinMaxScaler I was able to transformed my data that was too big to be either a 0 or 1, this was done with the use of the MinMaxScaler.
